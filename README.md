# Portfolio — Anghelo Alva

Personal portfolio of **Anghelo Alva**, full stack developer. End-to-end custom
software — from on-site requirements gathering to production support — for companies
in energy, mining, tourism and education.

Built on Next.js with a dark/tech aesthetic, Spanish (es-CL) UI, and a set of detailed
project case studies. The visual foundation comes from a wireframe-stage template
(neutral palette, hairline rails, WebGL dither shader); see
[`docs/design-system.md`](docs/design-system.md) for that reference.

## Stack

- **Next.js 16** (App Router) + **React 19**
- **TypeScript** (strict, `noUncheckedIndexedAccess`)
- **Tailwind CSS v4** with token-driven theming
- **Motion** (`motion/react`) with reduced-motion support
- **Lenis** smooth scroll (feature-flagged)
- **OGL** WebGL ASCII dither shader
- **next-themes** for dark mode
- **Radix UI / shadcn** primitives + **Hugeicons** / **Lucide** icons

## Getting started

## Editing the content

Almost everything that's personal lives in two files:

- **`lib/portfolio-data.ts`** — the projects and case studies. Each entry follows the
  `ProjectData` interface; flagship work adds a rich `caseStudy` (problem, solution,
  architecture, tech stack with rationale, metrics, timeline, testimonial). This data
  feeds the showcase, the community grid, and the `/proyectos/[id]` case-study pages.
- **`lib/metadata.ts`** — site name, description, keywords, and SEO/Open Graph config
  (`siteConfig`). The live domain is `https://angheloalva.dev`; the Open Graph image is
  generated by `app/opengraph-image.tsx`. An optional Twitter/X handle can still be added.

Other assets:

- `public/cv-anghelo-alva.pdf` — CV linked from the hero "Descargar CV" button.
- `public/logos/` — client/company logos for the logo marquee.
- `public/projects/` — project preview images.
- `app/icon.svg`, `app/apple-icon.svg` — favicon and Apple touch icon.

## Project structure

```
├── app/
│   ├── globals.css            # Design tokens & base styles
│   ├── layout.tsx             # Root layout with providers
│   ├── page.tsx               # Home one-pager
│   ├── proyectos/[id]/        # Per-project case-study page
│   ├── robots.ts              # Dynamic robots.txt
│   └── sitemap.ts             # Dynamic sitemap
├── components/
│   ├── header.tsx             # Sticky header with mobile menu
│   ├── showcase.tsx           # Project cards with morph overlay
│   ├── case-study.tsx         # Case-study page layout
│   ├── community.tsx          # Expandable project grid
│   ├── logo-marquee.tsx       # Client/company logo wall
│   ├── dither-shader.tsx      # WebGL ASCII dither shader (OGL)
│   ├── mockups/               # Per-project UI mockups + registry
│   ├── pricing.tsx            # Service tiers
│   ├── testimonials.tsx       # Testimonials
│   ├── faq.tsx                # FAQ accordion
│   ├── final-cta.tsx          # Final CTA
│   ├── footer.tsx             # Inverted footer card
│   └── ...                    # providers, reveal, theme-switch, ui/
├── lib/
│   ├── portfolio-data.ts      # Projects & case studies (main content)
│   ├── metadata.ts            # SEO / Open Graph config
│   ├── config.ts              # Feature flags
│   ├── motion.tsx             # Motion components & hooks
│   └── utils.ts
├── docs/
│   └── design-system.md       # Design tokens, shader & template reference
└── public/                    # CV, logos, project images, manifest
```
